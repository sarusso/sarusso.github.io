Projects

Timeseria

Timeseria is a time series processing library. It provides modules for data cleaning, resampling and reconstruction, as well as forecasting and anomaly detection models. [GitHub]

Pythings

Pythings is a quick and easy way of coding Internet of Things devices, directly from a web browser, in Python. It shines on micro-controllers, but if works nicely also on Linux boards [pythings.io] [GitHub]
