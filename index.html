Projects

Rosetta

Rosetta is a science platform for resource-intensive, interactive data analysis where user tasks are run as software containers. It can integrate with several computing resources from simple single-node setups to large HPC clusters. [Rosetta@OATS] [Paper]

Timeseria

Timeseria is an object-oriented time series processing library. It provides modules for data cleaning, resampling and reconstruction, as well as forecasting and anomaly detection models.
